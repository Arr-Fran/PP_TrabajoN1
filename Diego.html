Soy Diego Lautaro Abelleyra, Técnico Superior en Desarrollo de Software
Mi pasión por la tecnología y la programación me llevó a estudiar en el Instituto Superior N°46 "Domingo Guzmán Silva", donde he adquirido habilidades prácticas en desarrollo web, bases de datos y frameworks modernos.
A lo largo de los años, he trabajado en diversos proyectos que me han permitido perfeccionar mi capacidad de resolver problemas complejos y aportar soluciones eficientes.
Estoy siempre en búsqueda de nuevos retos que me permitan seguir creciendo y aportar valor a equipos multidisciplinarios.

HTML

Con HTML, he desarrollado una gran capacidad para estructurar y organizar contenido web de manera semántica. Utilizo etiquetas HTML5 para crear una base sólida que permita una fácil navegación y accesibilidad en las páginas web.

CSS

Con CSS, he aprendido a darle estilo y diseño a las páginas web, creando interfaces visualmente atractivas y responsivas. Utilizo flexbox, grid y otras técnicas avanzadas para asegurarme de que mis proyectos sean compatibles con diferentes dispositivos y tamaños de pantalla.

JavaScript

En JavaScript, he adquirido habilidades para hacer que las páginas web sean interactivas. Desde validación de formularios hasta la creación de aplicaciones dinámicas, domino la manipulación del DOM, eventos y AJAX, lo que me permite mejorar la experiencia del usuario de manera significativa.

NodeJS

Con Node.js, he trabajado en la creación de aplicaciones del lado del servidor, utilizando JavaScript para gestionar rutas, manejar bases de datos y crear APIs robustas. Me siento cómodo trabajando con express.js y otros frameworks para construir aplicaciones escalables.

SQL

En SQL, tengo experiencia gestionando bases de datos relacionales. Sé cómo crear consultas complejas, optimizar rendimiento y estructurar bases de datos de manera eficiente, lo que me permite almacenar, manipular y recuperar datos de manera efectiva en mis proyectos.
